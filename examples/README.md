# Odialang Examples

Welcome to the examples folder! These programs demonstrate various features of Odialang.

## 📂 Folder Structure

```
examples/
├── basics/           # Simple programs for beginners
│   ├── 01_hello.odia
│   ├── 02_variables.odia
│   ├── 03_operators.odia
│   ├── 04_conditionals.odia
│   ├── 05_while_loop.odia
│   ├── 06_for_loop.odia
│   ├── 07_strings.odia
│   └── 08_math.odia
├── intermediate/     # Programs with functions
│   ├── 01_functions.odia
│   ├── 02_recursion.odia
│   ├── 03_calculator.odia
│   ├── 04_temperature.odia
│   ├── 05_math_funcs.odia
│   ├── 06_scope.odia
│   └── 07_arrays.odia
└── advanced/         # Complex programs
    ├── 01_fizzbuzz.odia
    ├── 02_prime.odia
    ├── 03_fibonacci.odia
    ├── 04_factorial.odia
    ├── 05_grade_calculator.odia
    ├── 06_bank_account.odia
    ├── 07_sorting.odia
    └── 08_binary_search.odia
```

## 🚀 How to Run Examples

```bash
# Navigate to an example
# Run with:
odia examples/basics/01_hello.odia

# Or compile first:
odia compile examples/basics/01_hello.odia
node examples/basics/01_hello.js
```

## 📚 Learning Path

1. **Start with Basics** - Learn the fundamentals
2. **Practice with Intermediate** - Master functions
3. **Challenge with Advanced** - Build real programs

Happy coding! 🎉
